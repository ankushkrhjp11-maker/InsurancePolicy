# 🛡️ PolicyTrack Elite | Insurance Management System

PolicyTrack Elite ek premium Java Swing-based desktop application hai jo insurance policies aur staff authentication ko manage karne ke liye banayi gayi hai. Isme advanced UI/UX aur real-time features integrate kiye gaye hain.

## ✨ Key Features

* **Premium UI/UX:** Modern dark-themed interface jisme custom animations aur particle effects hain.
* **Secure Authentication:** Staff login system with password encryption.
* **AI Insurance Chatbot:** Google Gemini API (Flash 2.5) ka use karke Hinglish support ke sath chatbot.
* **Security Alerts:** Account par invalid login attempt hone par automatic email notifications (JavaMail API).
* **Live Dashboard:** Real-time clock aur system status monitoring.

## 🛠️ Tech Stack

* **Language:** Java (JDK 8+)
* **Frontend:** Java Swing, AWT (Custom Graphics2D)
* **Backend:** MySQL Database
* **APIs:** Google Gemini API, JavaMail API
* **Libraries:** JSON.org, MySQL Connector, Java Activation Framework (JAF)

## 🚀 How to Run

1.  **Database Setup:** `jdbc_swing` naam ka database banaye aur `users` table create karein.
2.  **API Key:** `Login.java` mein apni Gemini API Key update karein.
3.  **Dependencies:** Sari JAR files (MySQL Connector, JSON, Mail API) ko apne project path mein add karein.
4.  **Run:** `Login.java` file ko run karein.
